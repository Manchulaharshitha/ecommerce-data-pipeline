# eCommerce Data Pipeline# 🛒 End-to-End E-commerce Data Pipeline & Analytics

## 📌 Overview

This project demonstrates a complete data pipeline for an e-commerce system, covering data generation, cleaning, transformation, storage, analysis, and visualization.

---

## 🚀 Project Workflow

1. **Data Generation**

   * Synthetic data generated using Python (Faker)
   * Includes customers, products, and orders
   * Realistic timestamps spread across multiple months

2. **Data Cleaning & Transformation**

   * Handled missing values, duplicates
   * Standardized formats (email, phone, timestamps)
   * Ensured referential integrity

3. **Data Storage**

   * Uploaded cleaned data to Google BigQuery
   * Structured into:

     * orders (fact table)
     * customers (dimension table)
     * products (dimension table)

4. **Data Analysis (SQL)**

   * Revenue analysis
   * Customer segmentation
   * Product performance
   * Time-based trends

5. **Data Visualization**

   * Built interactive dashboard using Looker Studio
   * Includes KPIs, trends, and insights

---

## 📊 Key Insights

* Total Revenue and Order Volume
* Monthly Revenue Trends
* Top Performing Products
* State-wise Revenue Distribution
* Customer Behavior Analysis
